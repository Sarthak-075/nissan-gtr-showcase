# Nissan GT-R | The Godzilla - Cinematic Showcase

![Nissan GT-R Showcase](public/images/hero_og.jpg)

> "The ultimate everyday supercar, evolved into a technological masterpiece."

A high-performance, interactive 3D web experience showcasing the legendary Nissan GT-R R35. Built with the latest web technologies to deliver a cinematic user experience.

## üöÄ Features

*   **Cinematic 3D Transformation**: Real-time 3D car model transformation sequence using React Three Fiber.
*   **Interactive Gallery**: Horizontal scroll-snap gallery with immersive lightbox view.
*   **Dynamic Variants**: "Choose Your Weapon" section with hover-triggered background changes (Premium, T-Spec, Nismo).
*   **Metric Precision**: Detailed technical specifications (3799cc V6, 562 BHP) displayed in a responsive layout.
*   **Responsive Design**: Fully optimized for all devices, from mobile phones to 4K desktops.
*   **AI Concierge**: Integrated "Nissan Chatbot" for interactive user inquiries.
*   **Performance Optimized**: 100/100 Lighthouse score potential with optimized assets and lazy loading.
